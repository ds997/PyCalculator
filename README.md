# PyCalculator

This is a Python basic calculator with unit test cases.
The test data is stored in csv files.

CsvReader reads csv files and loads data for unit tests.

Functions it performs are as follows:

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Square
6. Square Root
